# Инструменты и MCP-серверы для производства сайтов EUROSFERA

> Список «навыков», которые усиливают изготовление сайта. ⭐ = рекомендую подключить.
> Установка MCP: `claude mcp add <name> ...` (делает Claude). Часть требует API-ключ/аккаунт (помечено 🔑).

## 🎨 Красота / вау-эффекты
- ⭐ **Context7 MCP** — свежая документация GSAP/Three.js/Tailwind → корректный современный код анимаций. Без ключа.
- ⭐ **Magic MCP (21st.dev)** 🔑 — генерит премиум-компоненты/секции по описанию. Бесплатный тариф.
- **Figma (Dev Mode) MCP** — макет из Figma → код 1:1 (нужен макет в Figma).
- **Playwright MCP** ✅ установлен — «глаза»: вижу/кликаю/скриншочу, проверяю вёрстку.
- **Генерация изображений** 🔑 (Replicate/Flux MCP) — уникальные визуалы/иконки вместо стоковых.
- **Spline** (spline.design) — 3D-сцены рисуются визуально → встраиваются `<spline-viewer>`. Нужен аккаунт.

## ⚙️ Код / функционал / контент
- **Pexels/Unsplash MCP** 🔑 — лицензионные фото по API (вместо ручного поиска).
- **Cloudinary MCP** 🔑 — оптимизация/трансформация изображений.
- **Lighthouse / PageSpeed MCP** — аудит скорости и SEO.
- **GitHub MCP** — управление репо/issue/деплой (сейчас делаю через `gh` CLI).

## 🚀 Публикация / SEO
- ⭐ **Netlify MCP** 🔑 — публикую сайт одной командой на бесплатный хостинг + живая ссылка (пока решается поддомен euro.s.bg).
- **Vercel MCP** 🔑 — то же, альтернатива.
- **Google Search Console** (не MCP) — регистрация сайта для индексации; нужен GA4-ID/верификация.

## Что уже встроено в наш сайт (без внешних MCP)
Three.js (глобус/объект/3D-машина model-viewer), GSAP, Lenis, vanilla-tilt, Typed.js, кастомный курсор,
кинематографичная заставка, цветные переходы + вспышка по направлениям, мультиязычность (Google Translate),
SEO (Schema.org/sitemap/robots). Всё локально в `lib/` (офлайн).
