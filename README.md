# EUROSFERA — сайт (v2, мультинаправления)

Статический сайт компании EUROSFERA LTD. Без бэкенда — чистые HTML/CSS/JS,
размещается на любом хостинге (SuperHosting.BG, и т.п.) простым копированием файлов.

## Страницы
| Файл | Раздел |
|------|--------|
| `index.html` | Главная (хаб всех направлений) |
| `goods.html` | Товары и оборудование |
| `logistics.html` | Логистика и склады |
| `it.html` | IT и лицензии ПО |
| `company.html` | Открытие компании в Болгарии + юр. сопровождение |
| `agro.html` | Удобрения и Агро (агроперлит, селитра, NPK) |
| `transfer.html` | Premium-трансфер (VIP-перевозки по Европе) |

Общие файлы: `enhance.css`, `enhance.js` (виджет, форма, счётчики, аналитика),
`img/`, `assets/`, `favicon.ico`.

## ⚙️ Настройка (всё в одном месте — начало `enhance.js`, блок `EURO_CONFIG`)

```js
const EURO_CONFIG = {
  whatsapp: "359892098460",
  telegram: "https://t.me/gagauz13",
  email:    "a.tukan@euro.s.bg",
  phone:    "+359 89 209 84 60",

  // КУДА ИДУТ ЗАЯВКИ:
  lead_webhook: "",        // 1) РЕКОМЕНДУЕТСЯ: URL n8n-webhook → Telegram-бот
  telegram_bot_token: "",  // 2) прямая отправка в Telegram (токен виден — только для теста)
  telegram_chat_id:   "",
  web3forms_key: "",       // 3) резерв: дублировать на email (web3forms.com)

  ga4_id: "",              // Google Analytics 4, напр. "G-XXXXXXXXXX"
};
```

### Заявки → Telegram-бот (рекомендуемая схема)
1. В n8n создать **Webhook** (POST) → нода **Telegram → Send Message**.
2. URL вебхука вставить в `lead_webhook`.
3. Форма шлёт JSON: `{ name, contact, direction, message, text, page }`,
   где `text` — уже готовое читаемое сообщение для отправки в чат.

Так токен бота остаётся на сервере (n8n), а не на публичном сайте — это безопасно.

### Аналитика
Создать ресурс GA4 на analytics.google.com → получить `G-…` → вставить в `ga4_id`.
Загружается только после согласия пользователя (баннер cookies, GDPR).

## Деплой
Скопировать ВСЕ файлы в `public_html` (или поддомен) на хостинге. Бэкенд не нужен.
